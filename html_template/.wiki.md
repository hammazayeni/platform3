# Project Summary
The Taekwondo Academy platform enhances communication and management for students, parents, and administrators. It provides a user-friendly interface for tracking attendance, managing schedules, and facilitating messaging between users. The platform aims to create a cohesive environment for learning and engagement in martial arts.

# Project Module Description
- **Messaging System**: Allows Admins, Parents, and Students to communicate in real-time.
- **Online Status Widget**: Displays the current online status of users on the Admin Dashboard.
- **Enhanced UI**: Improved visual elements including colors, icons, and layout for better user experience.

# File Description Inventory
- **admin-dashboard.html**: Interface for Admin users, featuring messaging and online user widgets.

- **index.html**: Main entry point for the application.
- **landing.html**: Introductory page for new users.

- **messages.js**: Handles the messaging functionalities.
- **online-status.js**: Manages the online status of users.

- **script.js**: General script for various functionalities across the platform.
- **settings.html**: User settings management interface.
- **student-dashboard.html**: Dashboard for students to track their progress and attendance.

# Usage
To get started with the Taekwondo Academy platform:

2. Build the project if necessary using your preferred build tool.
3. Run the application locally using your development server setup.
